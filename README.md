# AI Flashcard Generator

Upload lecture slides or notes (PDF/TXT/MD) and get back study flashcards generated by Claude.

## Setup

1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```

2. Create a `.env` file in the project root (copy from `.env.example`):
   ```bash
   cp .env.example .env
   ```
   Then paste your Anthropic API key into `.env`.
   Get one at https://console.anthropic.com/

3. Start the server:
   ```bash
   cd backend
   npm start
   ```

4. Open http://localhost:3000 in your browser.

## How it works

- **Frontend** (`frontend/`) — a single HTML page with an upload form, saved-deck list, and flippable flashcards.
- **Backend** (`backend/server.js`) — Express server with endpoints for generating, listing, loading, and deleting decks.
- **File parser** (`backend/services/fileParser.js`) — extracts text from PDF or plain-text uploads.
- **AI service** (`backend/services/aiService.js`) — sends the extracted text to Claude with a prompt asking for JSON flashcards.
- **Database** (`backend/db.js`) — SQLite (via `better-sqlite3`) with two tables (`decks`, `flashcards`). Persisted to `backend/flashcards.db`.

## API endpoints

| Method | Path | Description |
| --- | --- | --- |
| POST | `/api/generate` | Upload a file, generate flashcards, save as a new deck |
| GET | `/api/decks` | List all saved decks with card counts |
| GET | `/api/decks/:id` | Fetch a single deck with its flashcards |
| DELETE | `/api/decks/:id` | Delete a deck and its flashcards |
